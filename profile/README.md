## ⬇️ Download Options

### Option 1: Download Single PDF
1. Click on folder (e.g., `BSc-IT/Sem-I/2016-2017/`)
2. Click the PDF file
3. Click **"Download"** button (top right)

### Option 2: Download Entire Semester
```bash
git clone --depth 1 --filter=blob:none --sparse \
  https://github.com/mumbai-university-backup/pyqs.git
cd pyqs
git sparse-checkout set "BSc-IT/Sem-I"
# Just Sem-I downloads (~500MB)
```

### Option 3: Download Everything
```bash
git clone https://github.com/mumbai-university-backup/pyqs.git
# Full repo (~8GB)
```

### Option 4: Download with Browser (Easiest)
1. Go to [DownGit](https://downgit.github.io/)
2. Paste folder URL: `https://github.com/mumbai-university-backup/pyqs/tree/main/BSc-IT/Sem-I`
3. Click Download
4. Extract ZIP file
